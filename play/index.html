Never worry about a loss - this is not the same as ignoring it! Only this point matters. Disaster is an impostor.
It's not about being jaded, it's about being able to move on despite a setback. A loss is not a disaster. Let it go and focus on the next on. "Oh well, this has gone badly" but don't despair forever.
